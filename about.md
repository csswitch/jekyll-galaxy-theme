## Hello from the cosmos ✦

This blog is powered by **Galaxy Theme** — a deep-space Jekyll theme built by [csswitch](https://github.com/csswitch).

### Theme Features

- 🌌 **Canvas starfield** — real-time animated stars with parallax depth + twinkling
- 🌠 **Shooting stars** — CSS animated across the background
- 🔮 **Nebula glow layers** — animated radial gradients give depth
- 🌟 **Pulsar featured card** — your latest post gets a pulsing highlight
- ☄️ **Comet progress bar** — reads like a comet trail
- 📱 **Fully responsive** — constellation grid adapts to any screen
- ⚡ **Zero dependencies** — vanilla JS starfield engine (~5KB)
- 🐙 **GitHub Pages compatible** — deploy in seconds

### Configuration

```yaml
galaxy:
  nebula: "purple"      # purple | blue | gold | red | cyan
  stars: "normal"       # sparse | normal | dense
  shooting_stars: true
  comet_progress: true
```
